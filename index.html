Student Performance Analysis

Conducted an exploratory data analysis on student performance data to understand how social habits, parental education levels, and additional educational support influence academic outcomes. Utilized Python libraries such as Pandas, Matplotlib, and Seaborn to clean data, create visualizations, and identify key trends. Discovered that students who received additional support and had parents with higher education levels tended to perform better academically. Developed insights that could help educators tailor interventions for at-risk students.

Bellabeat Case Study

Performed a detailed case study using Bellabeat’s smart device data to explore user activity, sleep, and wellness patterns. Cleaned and transformed raw Fitbit data using R and SQL before conducting visual and statistical analysis in RStudio. Identified key user behavior trends and provided data-driven recommendations to enhance user engagement and support Bellabeat’s marketing strategy. Final deliverables included an executive summary and visual dashboard highlighting major findings.
